Advanced Probability Concepts

This activity will test your knowledge of key probability concepts, including permutations, combinations, and more.

Key Terms

  • Permutations: The number of ways to arrange items where the order of arrangement is important. Formula: $nPr = \frac{n!}{(n-r)!}$
  • Combinations: The number of ways to choose items from a group where order is not important. Formula: $nCr = \frac{n!}{r!(n-r)!}$
  • Conditional Probability: The likelihood of an event occurring, given that another event has already happened. Formula: $P(A|B) = \frac{P(A \cap B)}{P(B)}$
  • Mutually Exclusive Events: Two events that cannot occur at the same time. For these, $P(A \text{ and } B) = 0$.
