Introduction to Scratch/Python

Welcome, apprentice! Programming is like learning magic. You write special instructions (code) to command your computer. This set of instructions is called a program or a script.

The Two Schools of Magic

Just like wizards have different styles, there are different ways to code.

Scratch: The Magic of Blocks

Scratch is a visual language where you snap colorful blocks together. It's perfect for beginners because you don't have to worry about typing errors and you can see your results instantly on the 'Stage'.

Python: The Magic of Words

Python is a powerful, text-based language. You type commands using words and symbols. It's famous for being easy to read and is used by professionals to build websites, games, and much more!

Essential Magic Concepts

All programming, whether in Scratch or Python, uses the same core ideas.

üìú Sequence

A sequence is the order in which commands are followed. The computer reads your code from top to bottom, one line at a time, just like reading a book.

üîÑ Loops (Repetition Spells)

A loop is used to repeat a set of commands over and over. Instead of writing `move 10 steps` four times, you can use a loop to say `repeat 4 times: move 10 steps`. This saves time and keeps your code tidy.

üì¶ Variables (Magic Containers)

A variable is like a box or a container where you can store information. You can put a number, like a score, or text, like a player's name, inside a variable to use it later in your program.

ü§î Conditionals (Choice Spells)

Conditionals allow your program to make decisions. They use keywords like IF, ELSE, and THEN. For example: IF the score is greater than 10, THEN display "You win!".
