The periodic table is a chart that organizes all known chemical elements. It helps us understand their properties and predict how they will react.

Key Concepts

  • Atomic Number: The number of protons in an atom's nucleus. It's the unique ID for each element.
  • Groups: The vertical columns (1-18). Elements in the same group have similar chemical properties.
  • Periods: The horizontal rows (1-7).

How to Play the Games

  • Easy (Quiz Challenge): Answer multiple-choice questions about elements.
  • Medium (Find the Element): A treasure hunt on a simplified periodic table.
  • Hard (Property Match): A memory game matching elements to their properties.
