A flowchart is a visual diagram that shows the steps in a process. Think of it like a map that guides you from a starting point to a destination, showing all the paths and decisions along the way.

Why Use Flowcharts?

  • Clarity: They make complex processes easy to understand at a glance.
  • Planning: They help you think through all the steps needed to solve a problem before you start.
  • Communication: They are a simple way to show others how a process works.

The Basic Symbols

Flowcharts use standard symbols to represent different types of actions.

Start / End
An oval represents the beginning or end of the process.
Process / Action
A rectangle represents a step or an instruction to be carried out.
Input / Output
A parallelogram shows data that is being input or output.
Decision
A diamond represents a point where a "Yes/No" question is asked.
Flow Lines (Arrows)
Arrows connect the shapes and show the direction of the process.
