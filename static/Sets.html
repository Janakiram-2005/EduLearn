Introduction to Sets

What is a Set?

A set is a well-defined collection of distinct objects. The objects that make up a set are called its elements. Sets are usually denoted by capital letters, e.g., A = {1, 2, 3}.

Common Set Operations

  • Union (A ∪ B): The set of all elements which are in A, or in B, or in both.
  • Intersection (A ∩ B): The set of all elements which are common to both A and B.
  • Difference (A - B): The set of elements which are in A but not in B.
  • Complement (A'): The set of all elements in the universal set (U) that are not in A.
