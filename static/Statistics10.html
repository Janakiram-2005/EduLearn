Introduction to Statistics

Measures of Central Tendency

These are single values that attempt to describe a set of data by identifying the central position within that set.

  • Mean: The average of the data, found by summing all values and dividing by the number of values. For grouped data, it's Σ(fᵢxᵢ) / Σfᵢ.
  • Median: The middle value when the data is arranged in order. For grouped data, it is found using the formula L + [((n/2) - cf) / f] * h.
  • Mode: The value that appears most frequently. For grouped data, it is the class with the highest frequency.
