2. Control Flow

Control flow is the order in which statements are executed. It allows you to make decisions (if/else) and repeat actions (loops).

# An if-else statement
if age >= 18:
    print("Adult")
else:
    print("Minor")

# A for loop
for i in range(3):
    print(i) # Prints 0, 1, 2
