A database is an organized collection of data. Think of it as a digital filing cabinet, making information easy to find and manage.

Key Concepts

  • Data vs. Information: Data is raw facts (e.g., "16"), while information is data with context (e.g., "Age: 16").
  • Tables, Rows, Columns: A database uses tables to store data. Each row is a record, and each column is a field.
  • SQL: A language used to talk to databases (e.g., `SELECT * FROM Students`).

Game Modes

  • Easy (Sorting Showdown): Classify items as raw 'Data' or meaningful 'Information'.
  • Medium (Table Architect): Drag and drop columns to build a database table.
  • Hard (SQL Treasure Hunt): Use SQL queries to find the correct data treasure.
