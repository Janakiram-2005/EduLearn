1. Introduction to Computers

A computer is an electronic machine that takes in information (input), processes it, and gives a result (output). It's a powerful tool with abilities like speed, accuracy, and massive storage.

2. Parts of a Computer

Hardware vs. Software

A computer has two main parts:

  • Hardware: The physical parts you can touch, like the monitor, CPU, keyboard, and mouse.
  • Software: The set of instructions and programs that tell the hardware what to do, like Windows or a game.

Every computer task follows this simple cycle:

  1. Input: Giving the computer information (e.g., typing on a keyboard).
  2. Processing: The CPU (the computer's brain) works on the information.
  3. Output: The computer gives you the result (e.g., showing text on the monitor).

4. Memory and Storage

This is where a computer holds its data.

  • Primary Memory (RAM): Temporary memory for active tasks. It's fast but data is lost when the power is off.
  • Secondary Storage (Hard Disk): For long-term storage of files and programs.
