Understanding Linear Equations

A linear equation is an equation for a straight line. The main goal is to find the value of an unknown variable (like 'x') that makes the equation true. Think of it as finding a hidden number.

Key Principles

  • Isolate the Variable: Get the variable (x) by itself on one side.
  • Inverse Operations: Use opposites. Addition cancels subtraction; multiplication cancels division.
  • Keep it Balanced: Whatever you do to one side, you MUST do to the other side.

Step-by-Step Example

Let's solve the equation: 2x + 5 = 11

1. Undo Addition: Subtract 5 from both sides.

2x = 6

2. Undo Multiplication: Divide both sides by 2.

x = 3
