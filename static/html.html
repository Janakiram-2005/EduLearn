HTML (HyperText Markup Language) is the skeleton of a website. It uses special codes called tags to structure content.

Key Tags

  • <h1> for headings, <p> for paragraphs.
  • <img> to insert images, using the src attribute.
  • <a> to create links, using the href attribute.

Challenge Modes

  • Easy (Tag Triage): Identify the correct HTML tag for the job.
  • Medium (Structure Scramble): Drag and drop tags to build a valid webpage structure.
  • Hard (Code Cadet): Write HTML code to complete the mission.
