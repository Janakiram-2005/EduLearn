Understanding Polynomials

What is a Polynomial?

A polynomial is an expression made up of variables (like x), constants, and exponents, combined using addition, subtraction, and multiplication. A key rule is that variables cannot have negative or fractional exponents.

Types of Polynomials

We name polynomials based on the number of terms:

  • Monomial: One term (e.g., $5x$ or $-3y^2$).
  • Binomial: Two terms (e.g., $2x + 5$).
  • Trinomial: Three terms (e.g., $x^2 + 3x - 4$).

Adding and Subtracting

To add or subtract polynomials, you simply combine "like terms." Like terms have the exact same variable and exponent.

Example: $(3x^2 + 2x) + (5x^2 - x) = 8x^2 + x$.
