Algebraic Identities

What are Identities?

An algebraic identity is an equation that is true for all values of its variables. They are useful shortcuts for multiplying polynomials.

Standard Identities

Here are some of the most common identities you should know:

  • (a + b)² = a² + 2ab + b²
  • (a - b)² = a² - 2ab + b²
  • a² - b² = (a - b)(a + b)
  • (x + a)(x + b) = x² + (a + b)x + ab
