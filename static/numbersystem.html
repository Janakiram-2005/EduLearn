Understanding Number Systems

A number system is a way of writing and expressing numbers. We use the Decimal system daily, but computers use Binary.

1. Decimal (Base-10)

Uses 10 digits (0-9). The number 123 is 1*100 + 2*10 + 3*1.

2. Binary (Base-2)

Uses 2 digits (0 and 1). The binary 1101 is 1*8 + 1*4 + 0*2 + 1*1, which is 13 in decimal.

3. Hexadecimal (Base-16)

Uses 16 symbols (0-9 and A-F). The hex 1A is 1*16 + 10*1, which is 26 in decimal.

Conversion Examples

Binary → Decimal

To convert binary 1101 to decimal, multiply each digit by its power of 2: 8 + 4 + 0 + 1 = 13.

Decimal → Binary

To convert decimal 13 to binary, use repeated division by 2 and read the remainders upwards: 1101.
