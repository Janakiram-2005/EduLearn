Redox Reactions

LEO says GER!

This is a handy mnemonic for redox reactions:

  • Lose Electrons is Oxidation.
  • Gain Electrons is Reduction.

Oxidation & Reduction

  • Oxidation: A species is oxidized when it loses electrons, resulting in an increase in its oxidation state (e.g., Fe → Fe²⁺).
  • Reduction: A species is reduced when it gains electrons, resulting in a decrease in its oxidation state (e.g., Cu²⁺ → Cu).

Agents

In a redox reaction, one substance gets oxidized and the other gets reduced.

  • The Oxidizing Agent is the substance that gets reduced.
  • The Reducing Agent is the substance that gets oxidized.
