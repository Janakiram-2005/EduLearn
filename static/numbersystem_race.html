Number Systems

The Decimal System (Base-10)

The system we use every day. It has ten digits (0-9). The value of each digit depends on its place value (ones, tens, hundreds, etc.), which is a power of 10.

The Binary System (Base-2)

Computers use binary, which only has two digits: 0 and 1. The value of each digit is a power of 2. These digits are called bits.

Converting Binary to Decimal

To convert a binary number like 1011 to decimal, you add the place values for each '1': (1 × 8) + (0 × 4) + (1 × 2) + (1 × 1) = 11.
