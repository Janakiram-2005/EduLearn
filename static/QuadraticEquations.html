Solving Quadratic Equations

Standard Form

A quadratic equation's standard form is: ax² + bx + c = 0, where 'a' is not zero.

Methods of Solving

  • Factorization: Splitting the middle term.
  • Quadratic Formula: The universal formula to find the roots: x = [-b ± √(b² - 4ac)] / 2a.

The Discriminant

The value of D = b² - 4ac tells you about the roots:

  • If D > 0, there are two distinct real roots.
  • If D = 0, there is exactly one real root.
  • If D < 0, there are no real roots.
